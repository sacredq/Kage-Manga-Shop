# Kage Manga Shop

**Kage Manga Shop** is a responsive, modern front-end website built with HTML, CSS, and JavaScript. It showcases an anime and manga store with interactive features like login/sign-up forms, product purchase modals, and smooth animations.

## Features

- **Responsive Design**: Fully responsive layout suitable for desktop, tablet, and mobile devices.
- **Animated Effects**: Smooth scrolling and element animations using AOS (Animate On Scroll).
- **Interactive Modals**:
  - Login and Sign Up forms with validation
  - Buy Manga modal with dynamic product selection and order amount input
- **Clean UI/UX**: User-friendly interface with clear navigation and product presentation.
- **Static Product Catalog**: Includes manga titles with images, prices, and "Buy Now" buttons.

## Project Usage And Tools

- HTML5
- CSS3 (Flexbox, Grid, animations)
- JavaScript (DOM manipulation, event handling)
- [AOS.js](https://michalsnik.github.io/aos/) for scroll animations
- [Font Awesome](https://fontawesome.com) for icons
- Figma AI for design ideas
- Pinterest for Images
